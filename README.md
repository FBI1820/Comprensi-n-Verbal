# Test de Comprensión Verbal

Este proyecto es una aplicación web empresarial desarrollada con **OpenXava**, **Java** y **PostgreSQL**. Su objetivo es gestionar, aplicar y evaluar pruebas o test de comprensión verbal, herramientas fundamentales en el ámbito educativo y psicológico para medir la capacidad de un usuario para comprender, analizar e interpretar textos y estímulos lingüísticos.

## Requisitos Prévios e Instalación

Sigue estos pasos para configurar el entorno de desarrollo en tu máquina local utilizando **IntelliJ IDEA**.

### 1. Requisitos del Sistema
Asegúrate de tener instalado:
* **Java Development Kit (JDK):** Versión 11 o 17 (según la configuración de tu proyecto OpenXava).
* **PostgreSQL:** Versión 13 o superior.
* **IntelliJ IDEA:** Versión Community o Ultimate.

### 2. Configuración de la Base de Datos (PostgreSQL)
1. Abre tu terminal de PostgreSQL o una herramienta gráfica como pgAdmin.
